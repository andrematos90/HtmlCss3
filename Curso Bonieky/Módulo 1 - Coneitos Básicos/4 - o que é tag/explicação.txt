  TTAG

  As tags são usadas para informar ao navegador a estrutura do site. Ou seja: quando se escreve um código em HTML, as tags serão interpretadas pelo navegador, produzindo assim a estrutura e o conteúdo visual da página.

A principal característica das tags é estarem sempre dentro dos sinais de chevron (sinal de “maior que” e “menor que”), ou seja: < >.

As tags HTML são divididas em dois tipos: as que precisam de fechamento e as que não precisam de fechamento. As tags que precisam de fechamento possuem a sintaxe <tag> </tag>, já as que não precisam de fechamento possuem como estrutura <tag/>.

Além disso, uma mesma tag pode receber um ou mais atributos, que possuirá um valor que modifica sua estrutura ou funcionalidade.

As tags são códigos que definem toda a estrutura da página, tais como o seu tamanho,
 a fonte da letra, as cores,
 as quebras de linha e etc. 
A maioria dos elementos do documento HTML são compostos por uma estrutura de abertura e
 uma de fachamento,
 como <tag> e </tag>. Há também tags de estrutura única, 
como a  tag <br/> que realiza uma quebra de linha.
  
  
  
  
  
  Principais tags do HTML
Atualmente existem mais de 140 tags, mas algumas delas quase não são utilizadas. 
Dentre as mais utilizadas temos:

<html>: Define o início e o fim de um documento HTML.
<head>: Contém informações sobre o documento, como título e metadados.
<title>: Define o título do documento, que aparece na aba do navegador.
<body>: Contém o conteúdo principal do documento.
<h1> a <h6>: Define títulos de diferentes níveis, do maior (h1) ao menor (h6).
<p>: Define um parágrafo de texto.
<a>: Define um link para outra página ou recurso.
<img>: Insere uma imagem no documento.
<ul>: Define uma lista não ordenada.
<ol>: Define uma lista ordenada.
<li>: Define um item de lista.
<div>: Define uma seção genérica do documento, usada para agrupar elementos.
<span>: Define uma seção genérica do documento, usada para agrupar elementos em linha.
<table>: Define uma tabela.
<tr>: Define uma linha da tabela.
<td>: Define uma célula da tabela.
<th>: Define uma célula de cabeçalho da tabela.
<form>: Define um formulário para entrada de dados.
<input>: Define um campo de entrada, como texto, checkbox ou botão.
<select>: Define uma lista de opções selecionáveis.
<option>: Define uma opção de seleção dentro de um <select>.
<textarea>: Define um campo de entrada para múltiplas linhas de texto.
<label>: Define uma legenda para um campo de entrada.
<button>: Define um botão clicável.
<fieldset>: Define um grupo de campos dentro de um formulário.
<legend>: Define uma legenda para um <fieldset>.
<nav>: Define uma seção de navegação.
<header>: Define uma seção de cabeçalho do documento.
<footer>: Define uma seção de rodapé do documento.
<aside>: Define uma seção de conteúdo adicional relacionado ao conteúdo principal.
<section>: Define uma seção genérica do documento.
<article>: Define um artigo independente dentro do documento.
<main>: Define a seção principal do documento.
<time>: Define um trecho de tempo, como uma data ou hora.
<progress>: Define uma barra de progresso.
<meter>: Define um medidor, como para avaliar o nível de uma bateria ou temperatura.
<audio>: Define um elemento de áudio.
<video>: Define um elemento de vídeo.
<source>: Define uma fonte de mídia para o elemento <audio> ou <video>.
<iframe>: Cria uma janela interna no documento para exibir outra página ou recurso.
<script>: Insere um script JavaScript no documento.
<noscript>: Define o conteúdo a ser exibido quando o JavaScript está desativado no navegador.
<style>: Define estilos CSS para o documento ou para uma seção específica.
<meta>: Define informações sobre o documento, como codificação de caracteres, descrição, palavras-chave e autor.
<link>: Define um link para um arquivo externo, como uma folha de estilos CSS ou um ícone.
<hr>: Insere uma linha horizontal no documento.
<br>: Insere uma quebra de linha dentro de um texto.
<em>: Define um texto em destaque ou ênfase.
<strong>: Define um texto em negrito para maior ênfase.
<i>: Define um texto em itálico.
<u>: Define um texto sublinhado.
<s>: Define um texto riscado para indicar exclusão.
<sup>: Define um texto com sobrescrito, como uma notação matemática ou de nota de rodapé.
<sub>: Define um texto com subscrito, como uma notação química ou de nota de rodapé.
