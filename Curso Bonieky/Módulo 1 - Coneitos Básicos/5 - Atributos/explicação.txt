Atributos
Os atributos são usados para personalizar as tags, modificando sua estrutura ou funcionalidade. Igualmente, os atributos são utilizados para atribuir uma classe ou id a um elemento.

A maioria das tags tem seus próprios atributos. Contudo, existem alguns atributos genéricos que podem ser utilizados na maioria das tags HTML, vamos estudá-los:

class=”…“ – Atribui uma classe ao elemento (uma classe pode ser utilizada para um ou mais elementos);
id=”…“ – Atribui um id ao elemento (um id deve ser único, ou seja atribuído a um único elemento);
style=”…” – Permite incluir elementos CSS (estilos) dentro da tag;
lang=”…” – Define o idioma principal do elemento;
title=”…” – Define o título do elemento;
alt=”…” – Define um texto alternativo e, por isso, é muito utilizado em imagens, auxilia nas práticas de SEO;
hidden – Oculta o elemento;
align=”…” – Permite definir o padrão de alinhamento desse elemento, como por exemplo: right, center, left e justify;
width=”…” – Define uma largura para o elemento;
height=”…” – Define uma altura para o elemento.
Essas são os principais atributos, porém existem diversos outros que devem ser estudados e podem ser utilizados no seu código.

-------------------------------------------------------------------------------------------------------------

Em HTML, os atributos são utilizados para fornecer informações adicionais sobre os elementos (tags). Eles são usados para modificar o comportamento ou a aparência dos elementos, definir valores, especificar conexões ou fornecer metadados relevantes.

Os atributos são adicionados aos elementos como pares de nome/valor dentro das tags. Eles fornecem informações extras ou instruções específicas que afetam o elemento. Aqui está um exemplo básico de como um atributo é definido em uma tag HTML:

<tag atributo="valor">Conteúdo</tag>


Nesse exemplo, "atributo" é o nome do atributo, e "valor" é o valor atribuído a esse atributo.

Alguns elementos HTML têm atributos específicos associados a eles. Por exemplo, a tag <a> (âncora) possui o atributo "href" que especifica o URL de destino do link, enquanto a tag <img> (imagem) tem o atributo "src" que indica o caminho para a imagem a ser exibida.

Aqui está um exemplo de uso desses atributos:

<a href="https://www.exemplo.com">Link para Exemplo</a>

<img src="caminho/para/imagem.jpg" alt="Descrição da Imagem">


Os atributos também podem ser usados para fornecer informações adicionais para fins de acessibilidade, formatação de estilo (usando CSS) ou interação com JavaScript. A disponibilidade e o uso de atributos variam de acordo com o elemento e a versão do HTML sendo usada.

É importante observar que alguns atributos foram descontinuados ou substituídos em versões mais recentes do HTML, portanto, é recomendado consultar as especificações oficiais ou referências atualizadas para obter informações precisas sobre os atributos disponíveis e seu uso adequado.
