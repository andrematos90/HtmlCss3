                               FRONT-END

 Front-end é a parte de um site ou aplicativo que o usuário final interage diretamente. É o conjunto de tecnologias e técnicas usadas para criar a interface do usuário, incluindo design, layout, interação e funcionalidade.

O front-end é construído usando principalmente três tecnologias: HTML, CSS e JavaScript. O HTML é usado para criar a estrutura básica da página, o CSS é usado para estilizar a página e o JavaScript é usado para adicionar interatividade e dinamismo ao front-end.

Alguns exemplos de elementos de front-end em um site incluem:

Layout e design
Menus de navegação
Botões, formulários e campos de entrada
Imagens e vídeos
Animações e efeitos de transição
Funcionalidades interativas, como menus dropdown e carrosséis de imagens
O desenvolvimento front-end requer habilidades em design, programação, arquitetura de informação, usabilidade e acessibilidade. Um bom desenvolvedor front-end deve ser capaz de criar interfaces atraentes, funcionais e responsivas, que sejam acessíveis em diferentes dispositivos e navegadores
